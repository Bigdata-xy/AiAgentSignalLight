# Engineering Standards

- Keep Core free of WPF, PowerShell, browser, and filesystem-specific assumptions.
- Keep adapters thin and replaceable.
- Use UTF-8 for every file.
- Use JSON as the MVP storage format.
- Treat diagnostics as a first-class feature.
- Preserve the traffic-light UI as the primary visual surface.
- Do not bind product names, storage directories, or protocols to Codex.
